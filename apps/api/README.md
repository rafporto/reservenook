# API App

This is the real Spring Boot scaffold for the Reservenook backend.

Current baseline:

- Kotlin
- Spring Boot
- Spring Web
- Spring Security
- Spring Data JPA
- Flyway
- PostgreSQL driver
- Testcontainers-ready test dependencies

Primary directories:

- `src/main/kotlin/com/reservenook`
- `src/main/resources`
- `src/test/kotlin/com/reservenook`
- `src/test/resources`
