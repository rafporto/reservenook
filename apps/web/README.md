# Web App

This is the real Next.js scaffold for the Reservenook frontend.

Current baseline:

- App Router
- TypeScript
- MUI theme provider
- TanStack Query provider
- next-intl dependency reserved for i18n setup
- Vitest baseline for frontend tests

Primary directories:

- `src/app`
- `src/components`
- `src/features`
- `src/lib`
- `public`
